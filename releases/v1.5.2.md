## New Features

* Official release binaries now built with Go 1.13+, enabling TLS 1.3 by
  default.
* Keystore flags exposed as environment variables (#250).

## Bug Fixes

* Fixed TLS 1.3/PKCS11 issues (#271).
* Fixed ECC/PKCS11 compatibility (#257).
* Resolved client mode status availability bug with auth disabled (#268).
* Updated external dependencies, eliminating libtool/libltdl runtime
  requirement.
