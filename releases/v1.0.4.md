## Notable Changes

* Better child termination logic. Ensure we terminate child if shutdown times
  out (#92).
* Make `--cacert` flag optional, use system roots by default if flag is not
  specified (#93).

Due to the change in #93, Ghostunnel now requires Go 1.7 for building.
