Release candidate for v1.10.0.

## Highlights

* PROXY protocol v2 now includes TLS metadata TLVs, forwarding information
  about the TLS connection to the backend (#705).
* Fixed macOS keychain memory leaks (CFObject) and data races in lazy
  initialization (#699, #700).
* Fixed Windows certificate store search order (#704).
* Replaced `github.com/pkg/errors` with stdlib `errors` and `fmt` (#684).
* Removed `certigo` dependency (#664), switched to `smallstep/pkcs7` (#663).

## Changes

* PROXY protocol v2 augmented with TLS metadata TLVs (#705).
* Fixed CFObject leaks in macOS CertificateChain (#699).
* Fixed data races in macOS keystore lazy init (#699).
* Fixed Windows cert store search order (#704).
* Added Windows certificate store integration tests (#702).
* Replaced `unsafe.Pointer` with `atomic.Pointer[T]` in certloader (#679).
* Fixed C string leak in `launchdSocket` (#656).
* Fixed C array leak in `getProviderParam` on Windows (#656).

## Minor Changes

* Launched project website (#680, #685).
* Comprehensive documentation improvements (#657, #659, #707).
* Parallelized integration tests with dynamic port allocation (#662).
* Added Windows CI integration tests (#695).
* Extensive test infrastructure improvements (#662, #696, #697, #703).
* Python test linting and code quality fixes (#687, #688, #689).
* Removed dead code, un-exported unused symbols (#665).
* Dependency upgrades (#663, #674, dependabot PRs).
* Migrated to `docker buildx build` (#660).
* Applied `go fix` with Go 1.26 (#658).
