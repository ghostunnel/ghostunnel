## New Features

* Add optional `/_shutdown` endpoint on status port for terminating via HTTP
  POST. Can be enabled via the `--enable-shutdown` flag (#466).
* On Windows, add support for using `LOCAL_MACHINE` and `CURRENT_SERVICE` key
  stores. Was previously limited to the `CURRENT_USER` key store (#476).

## Bug Fixes

* Better landlock rule processing to handle flags that can be both `host:port`
  or URLs correctly (#475).
