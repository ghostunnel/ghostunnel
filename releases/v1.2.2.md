## Notable Changes

* Added experimental support for the macOS keychain via certstore (requires
  build tag, disabled by default).
* Docker image updated with multistage builds to reduce size, removing Go
  toolchain from the image.
* Added support for the Prometheus metrics format in the `/_metrics` endpoint,
  alongside JSON.

## Other

* Code modularization and cleanup improvements.
* Documentation enhancements.
